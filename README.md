## вљ™пёЏ Configuration

Edit these values at the top of `main.py`:

| Variable | Default | Description |
|---|---|---|
| `CONFIDENCE` | `0.5` | Detection threshold (0.0вЂ“1.0) |
| `DETECT_EVERY_N` | `3` | Run AI every N frames |
| `INFER_SIZE` | `320` | Input size for model (px) |
| `DISAPPEAR_DELAY` | `2` | Seconds before alarm stops |
| `PHONE_CLASS_ID` | `77` | COCO class ID for cell phone |

**Tuning tips:**

- Fewer false positives в†’ increase `CONFIDENCE` to `0.6`вЂ“`0.65`
- Faster display в†’ increase `DETECT_EVERY_N` to `4`вЂ“`5`
- More responsive detection в†’ decrease `DETECT_EVERY_N` to `1`вЂ“`2`

## вќ“ Troubleshooting

**Camera not opening:**
```bash
# Linux вЂ” check available cameras
ls /dev/video*

# Try a different index in main.py
cap = cv2.VideoCapture(1)
```

**No audio alarm:**
- Make sure `Assets/` folder has a supported audio file
- Re-run `setup.py` to auto-detect it

**Slow detection:**
- Increase `DETECT_EVERY_N` to `4` or `5`
- Lower `INFER_SIZE` to `224`
- Enable OpenCL in `main.py` (uncomment the backend lines)

**ModuleNotFoundError:**
```bash
# Make sure venv is activated
source .venv/bin/activate    # Linux/macOS
.venv\Scripts\activate       # Windows
```
